
## document.querySelector()



返回找到的第一个节点



```js



const element1 = document.querySelector('p')



```



## document.querySelectorAll()



返回所有找到的节点组成的类数组对象



```js



const elements1 = document.querySelector('p')

const elements2 = document.querySelector('.aaa')

```



## 快捷选择元素



document.head	`<head>`元素

document.body	`<body>`元素

document.title	文档标题

document.images	文档中的`<img>`节点组成的类数组

document.links	文档中的`<a>`节点组成的类数组
